## Правила игры:
Игроки управляют с помощью клавиш ракеткой и отбивают мяч. Проиграет тот, кто не пропустит мяч.

## Rules of the Game:
Players use the keys to control a paddle and hit a ball. The one who doesn't miss the ball loses.
