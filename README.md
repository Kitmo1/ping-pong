# pin-pong
Repository for my project "pin-pong"
Это прототип игры "ping-pong". Используется язык программирование "Python". Для игры нужно два игрока. Чтобы начать играть в игру, скачайте картинки(png)

This is a prototype of the game "ping pong." The programming language is Python. The game requires two players. To start playing the game, download the images(png)

## Правила игры:
Игроки управляют с помощью клавиш ракеткой и отбивают мяч. Проиграет тот, кто пропустит мяч.

## Rules of the Game:
Players use the keys to control a racket and hit the ball. The one who misses the ball loses.
